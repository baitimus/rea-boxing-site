# Video Assets

## Required Video
Place your boxing workout video here:
- **Filename**: `boxing-workout.mp4` and/or `boxing-workout.webm`
- **Recommended specs**:
  - Resolution: 1920x1080 (Full HD) minimum
  - Format: MP4 (H.264) and/or WebM (VP9)
  - Duration: 10-30 seconds looping clip
  - Content: Someone doing boxing/workout movements
  - Style: Professional, clean, well-lit
  - Aspect Ratio: 16:9

## Notes
- The video will be used as the hero section background
- It will have a navy overlay (70% opacity) applied
- Ensure the video has rights for web use
- Compress for web delivery (aim for under 5MB)
